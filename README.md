# Astro Starter

This project has been reset to a minimal Astro starter site.

## Commands

All commands are run from the project root:

```sh
npm install
npm run dev
npm run build
npm run preview
```

## Project Structure

```text
/
├── public/
├── src/
│   ├── components/
│   ├── layouts/
│   ├── pages/
│   └── styles/
├── astro.config.mjs
├── package.json
└── tsconfig.json
```

The homepage lives at `src/pages/index.astro`.
